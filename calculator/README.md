# Python Calculator

This is a simple calculator that supports basic math operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Elevation
- Square root
- Residue from division

## How to use
1. Run the calculator.py file.
2. Follow the on-screen instructions.

## Installation
The program requires Python 3.x.
